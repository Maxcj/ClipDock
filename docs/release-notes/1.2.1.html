ClipDock 1.2.1

Release notes for the Sparkle-powered update channel.

中文

本次更新

ClipDock 1.2.1 正式接入 Sparkle 自动更新通道，增强了应用分发与版本管理能力。

  • 正式接入 Sparkle 自动更新，支持检查、下载、校验与安装更新。
  • 设置页新增独立的 Updates 标签页，支持手动检查更新。
  • 支持忽略指定版本，并可在后续重新显示该版本。
  • 通过 GitHub Pages 发布 appcast，GitHub Releases 承载安装包下载。
English

What’s New

ClipDock 1.2.1 formally introduces the Sparkle update channel to improve distribution and version management.

  • Sparkle automatic updates are now integrated for update checks, download, verification, and installation.
  • A dedicated Updates tab has been added to Settings for manual update checks.
  • Users can ignore a version and bring it back later when needed.
  • The appcast is published through GitHub Pages, while GitHub Releases hosts the installer archive.
