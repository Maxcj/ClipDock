ClipDock 1.2.1

Release notes for the Sparkle-powered update channel.

What's new

  • Added Sparkle update support with automatic checks, download, verification, and installation.
  • Added a dedicated Updates tab in Settings with a manual Check for Updates action.
  • Added support for ignoring a version and showing it again later.
  • Configured GitHub Pages appcast publishing for https://maxcj.github.io/ClipDock/appcast.xml.
