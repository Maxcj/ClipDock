# ClipDock 0.9.3

ClipDock 0.9.3 brings color recognition, stronger code snippet handling, and a round of UI and behavior refinements.

### 本次更新

- 新增剪贴板颜色识别。
- 颜色详情保留原始复制内容，同时在副标题和色块区域展示标准化 HEX 值。
- 精简颜色复制操作，减少重复按钮。
- 优化代码片段识别与展示体验。
- 让分类标签更紧凑。
- 优化链接展示，右侧显示网站 favicon，来源恢复为原始复制 APP。
- 修复不同 APP 复制相同内容时的重复记录问题。
- 修复取消 Pin 后回到原始排序位置的问题。
- 优化窗口激活行为，减少启动和菜单栏相关警告。
- 支持 Sparkle gentle reminders，降低后台更新提醒噪音。

### 说明

- 更新地址：`https://maxcj.github.io/ClipDock/appcast.xml`
- 发布资产：`ClipDock-0.9.3-macOS-universal.zip`

### Highlights

- Added clipboard color recognition.
- Color details now keep the original copied text while showing normalized HEX values in the subtitle and swatch area.
- Streamlined color copy actions to remove redundant buttons.
- Improved code snippet recognition and presentation.
- Made category chips more compact.
- Refined link presentation with website favicon previews and source app restoration.
- Fixed duplicate records when the same content is copied from different apps.
- Restored the original ordering when unpinning an item.
- Improved window activation behavior and reduced startup and menu-bar warnings.
- Supported Sparkle gentle reminders to reduce background update noise.
