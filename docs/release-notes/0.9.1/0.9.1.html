ClipDock 0.9.1

Release notes for the Sparkle-powered update channel.

中文

本次更新

ClipDock 0.9.1 主要聚焦在发布信息整理与更新流程稳定性上。

  • release note 的目录结构已按版本号分目录整理。
  • appcast 现在只保留最近的发布条目，让订阅内容更干净。
  • 手动检查更新在没有新版本时也会给出明确提示。
  • Sparkle 的更新体验继续保持为“检查更新 + 展示说明 + 打开下载页”。
English

What’s New

ClipDock 0.9.1 focuses on release-feed cleanup and update-flow stability.

  • The release-note layout is now organized by version directory.
  • The appcast now keeps only the most recent release entries.
  • Manual update checks now show a clear message when no new version is available.
  • Sparkle continues to provide the “check for updates + show release notes + open download page” flow.
