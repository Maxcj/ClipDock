Release Notes

ClipDock 0.9.4

Category management gets a usability pass in this release.

  • Left category cards are taller for better readability.
  • Categories can now show up to 3 items per card.
  • Custom category edit and delete buttons use a consistent size.
  • Add category cancel and save buttons now match in size.
  • Custom categories support custom colors.
  • System categories and custom categories can be reordered together.
