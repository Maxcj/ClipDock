# ClipDock 1.2.2

ClipDock 1.2.2 focuses on UI polish, layout stability, and a smoother release workflow.

## 中文

### 本次更新

- 固定主窗口和设置窗口尺寸，避免窗口缩放引发布局抖动。
- 精简右侧详情区域，图片和文件类型不再额外显示路径信息。
- 调整主窗口和设置窗口的初始尺寸，让固定布局下的视觉密度更均衡。
- 图片历史默认开启，文件历史默认关闭，符合更常见的使用习惯。
- 继续整理构建与发布脚本，为后续版本发布保留更清晰的流程。

### 说明

- 更新地址：`https://maxcj.github.io/ClipDock/appcast.xml`
- 发布资产：`ClipDock-1.2.2-macOS-universal.zip`

## English

### Highlights

- The main window and Settings window now use fixed sizes to avoid resize-related layout jitter.
- The right-side detail area has been simplified so image and file entries no longer show path text there.
- Initial window sizes were tuned to better fit the fixed-layout experience.
- Image history remains enabled by default, while file history stays disabled by default.
- Build and release scripts have been organized to keep future releases cleaner and easier to repeat.

### Notes

- Appcast URL: `https://maxcj.github.io/ClipDock/appcast.xml`
- Release asset: `ClipDock-1.2.2-macOS-universal.zip`
