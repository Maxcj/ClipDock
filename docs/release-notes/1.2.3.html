ClipDock 1.2.3

Release notes for the Sparkle-powered update channel.

中文

本次更新

ClipDock 1.2.3 继续完善更新流程，并延续了前一版的界面与发布优化。

  • 检查更新现在会在发现新版本时提示用户前往下载，而不会再走自动安装流程。
  • 如果当前没有可用更新，也会明确提示用户“当前已经是最新版本”。
  • 保留“忽略某个版本”的能力，方便跳过不想安装的更新。
  • 继续沿用前一版的窗口固定尺寸、详情区域精简和历史保留设置。
English

What’s New

ClipDock 1.2.3 continues the update-flow polish and keeps the recent UI and release improvements in place.

  • Checking for updates now prompts users to download the new release instead of launching the automatic installer flow.
  • When no update is available, ClipDock now clearly says that you are already on the latest version.
  • The ignored-version workflow remains available for skipping releases you do not want to install.
  • This release keeps the fixed-window layout, streamlined detail panels, and history retention defaults from the previous version.
