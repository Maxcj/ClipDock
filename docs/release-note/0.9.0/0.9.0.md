# ClipDock 0.9.0

ClipDock 0.9.0 focuses on update distribution, release automation, and a cleaner settings experience.

## 中文

### 本次更新

- Sparkle 已切换为 Swift Package 集成方式，便于后续维护和 GitHub Action 发布。
- 发布流程已接入 GitHub Actions，可自动构建、生成 Release 资产并更新 appcast。
- 更新设置页新增自动检查更新周期，只有开启自动检查更新后才可编辑。
- 应用启动时会自动触发一次更新检查，帮助用户更快发现新版本。
- 更新说明窗口的顶部重复关闭按钮已移除，保留底部关闭按钮即可。

### 说明

- 更新地址：`https://maxcj.github.io/ClipDock/appcast.xml`
- 发布资产：`ClipDock-0.9.0-macOS-universal.zip`

## English

### Highlights

- Sparkle is now integrated through Swift Package Manager, which makes maintenance and GitHub Actions publishing cleaner.
- The release flow is now connected to GitHub Actions so builds, release assets, and appcast updates can be automated.
- The Updates settings page now includes an automatic-check interval, and it can only be edited when automatic checking is enabled.
- ClipDock now performs one update check when the app launches so users can discover new releases sooner.
- The duplicate top-right close button has been removed from the release notes window, leaving the bottom close button as the only close action.

### Notes

- Appcast URL: `https://maxcj.github.io/ClipDock/appcast.xml`
- Release asset: `ClipDock-0.9.0-macOS-universal.zip`
