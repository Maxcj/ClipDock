# ClipDock 0.9.6

ClipDock 0.9.6 进一步提升了剪贴板内容识别稳定性，并让代码与颜色相关元数据在重启后继续可用，同时精简了分类管理体验。

### 本次更新

- 改进纯文本、RTF 和 HTML 剪贴内容的解析，提升代码识别稳定性。
- 持久化代码语言、代码行数以及颜色元数据，重启后依然可用。
- 精简分类排序逻辑，并统一相关文案。

ClipDock 0.9.6 further improves clipboard content detection, keeps code and color metadata available after relaunch, and streamlines category management.

### What’s New

- Improve plain text, RTF, and HTML clipboard parsing for more reliable code detection.
- Persist code language, line count, and color metadata so they remain available after relaunch.
- Simplify category ordering logic and align related copy.
