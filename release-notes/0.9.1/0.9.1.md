# ClipDock 0.9.1

ClipDock 0.9.1 is a release focused on release-feed cleanup and update-flow stability.

## 中文

### 本次更新

- 调整了 release note 的目录结构，后续版本会按版本号分目录管理。
- appcast 只保留最近的发布条目，让更新订阅更干净。
- 手动检查更新现在在没有新版本时也会给出明确提示。
- Sparkle 更新说明与下载流程保持为“检查更新 + 展示说明 + 打开下载页”的体验。

### 说明

- 更新地址：`https://maxcj.github.io/ClipDock/appcast.xml`
- 发布资产：`ClipDock-0.9.1-macOS-universal.zip`

## English

### Highlights

- The release-note layout now groups files by version, keeping future releases organized by version directory.
- The appcast now keeps only the most recent release entries, which makes the feed cleaner.
- Manual update checks now show a clear message when no new version is available.
- Sparkle continues to provide the “check for updates + show release notes + open download page” experience.

### Notes

- Appcast URL: `https://maxcj.github.io/ClipDock/appcast.xml`
- Release asset: `ClipDock-0.9.1-macOS-universal.zip`
