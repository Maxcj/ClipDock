ClipDock 1.1.0-beta

Release notes for the Sparkle-powered update channel.

本次更新

ClipDock 1.1.0-beta 进一步完善了双更新通道，正式版和 beta 版可以通过设置切换各自的更新源，并由发布流程按标签自动生成对应的 appcast。

  • 新增正式版和 beta 双更新通道
  • 可在设置中切换更新源，按需接收 beta 更新
  • 发布流程会按 tag 类型生成对应的 appcast

What’s New

ClipDock 1.1.0-beta further refines the dual update-channel setup. Release and beta builds can now switch feeds from Settings, and the release workflow publishes the matching appcast based on tag type.

  • Added separate release and beta update channels
  • Switch feeds from Settings to opt into beta updates
  • Release automation now generates the matching appcast by tag type
