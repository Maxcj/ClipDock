ClipDock 1.1.1-beta3

Release notes for the Sparkle-powered update channel.

本次更新

ClipDock 1.1.1-beta3 进一步优化了 beta 更新体验，同时继续保留设置中的自定义敏感规则，方便快速调整隐私策略。

  • 继续支持在设置中直接管理自定义敏感规则
  • 保留关键词和正则两种规则类型，以及“包含”或“完全相等”的匹配方式
  • 更新设置里的 Sparkle 用户流程入口更直接，便于测试标准更新界面

What’s New

ClipDock 1.1.1-beta3 improves the beta update experience and keeps the custom sensitive rules workflow available in Settings for quick privacy tuning.

  • Custom sensitive rules remain available directly in Settings
  • Keyword and regex rule types are still supported, including contains and exact-match modes
  • The Sparkle user-flow entry in Settings is more direct when testing the standard update UI
