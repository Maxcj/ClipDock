# ClipDock 1.1.1-beta2

ClipDock 1.1.1-beta2 keeps the custom sensitive rules workflow in place and refines the update experience so beta testing feels smoother and more direct.

### 本次更新

- 继续支持自定义敏感规则，可在设置中直接管理
- 保留关键词和正则两种规则类型，以及“包含”或“完全相等”的匹配方式
- 更新设置中的 Sparkle 用户流程入口更直观，便于测试标准更新界面

ClipDock 1.1.1-beta2 keeps the custom sensitive rules workflow in place and refines the update experience so beta testing feels smoother and more direct.

### What’s New

- Custom sensitive rules are still available directly in Settings
- Keyword and regex rules remain supported, including contains and exact-match modes
- The Sparkle user-flow entry in Settings is easier to use when testing the standard update UI
