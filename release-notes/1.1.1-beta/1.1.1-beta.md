# ClipDock 1.1.1-beta

ClipDock 1.1.1-beta adds custom sensitive rules to the privacy settings, giving you finer control over which clipboard items should never be saved.

### 本次更新

- 新增自定义敏感规则，可在设置中直接管理
- 支持关键词和正则两种规则类型
- 关键词规则可选择“包含”或“完全相等”匹配方式
- 可为规则设置适用范围，并支持启用或禁用

ClipDock 1.1.1-beta adds custom sensitive rules to Privacy Settings, giving you finer control over which clipboard items should never be saved.

### What’s New

- Added custom sensitive rules directly in Settings
- Support both keyword and regex rule types
- Keyword rules can match by contains or exact match
- Rules can be scoped and enabled or disabled
