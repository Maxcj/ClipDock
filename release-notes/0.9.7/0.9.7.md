# ClipDock 0.9.7

ClipDock 0.9.7 修复了 Core Data 版本切换带来的复制崩溃，并进一步加固了剪贴板元数据访问，提升了整体稳定性。

### 本次更新

- 修正 Core Data 当前模型版本，避免复制时因旧模型缺少新字段而崩溃。
- 移除对剪贴板记录元数据的脆弱访问方式，降低运行时异常风险。
- 保留版本控制能力，同时让模型演进后的数据读取保持稳定。

ClipDock 0.9.7 fixes the Core Data version mismatch that could crash during clipboard capture and further hardens metadata access for better stability.

### What’s New

- Fix the Core Data current version so clipboard capture no longer crashes on missing model fields.
- Replace fragile metadata access patterns to reduce runtime exception risk.
- Keep versioned models enabled while making evolved data reads stable.
